Easily insert text symbols or emojis into your posts or pages. This works with the Gutenberg editor.

== Description ==
 
This plugin will add a button to the Gutenberg editor for inserting text symbols or emojis into your WordPress posts or pages.

Made by: [MockoFun Online Graphic Designer](https://www.mockofun.com)

><strong>Features</strong>

>- Over 1000 icons, symbols and emojis included
>- Insert Icons Directly Into WordPress Posts & Pages using the Gutenberg editor
>- No need to write a single line of code
>- Actively under development

== Installation ==

To install the Insert Text Symbols plugin do this:

1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the wp-content/plugins/ directory of your WordPress installation.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the Gutenberg editor new button for inserting symbols
